# JavaCodeTest
1. This is a Maven project. 

2. Test cases are in AppTest.java class under the Package Nodapo.JavaCode. Run directly AppTest.java by right clicking and Run as -> JUnit Test

3. DataStore.java class under the Package Nodapo.JavaCode.DataStore supplies the data. 

4. CustomerService contains method for customer related processing

5. ShopService contains methods for shop related processing

6. The functionality of Customer could be further extended like after purchase book deduct his purse, add a book to the customer's list of books etc. but these are not covered in the requirements.
